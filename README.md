# 🎮 Guess The Number Game (Multiplayer)

A multiplayer number guessing game developed in C.

## 📌 Features

- Two-player gameplay
- Random number generation
- Winner based on minimum guesses
- User-friendly console interface
- Built using C programming

## 🛠️ Technologies Used

- C Programming
- Standard C Library
- Windows Console

## ▶️ How to Run

1. Compile the program.
2. Run the executable.
3. Follow the on-screen instructions.
4. Both players try to guess their random number.
5. The player with fewer attempts wins.
